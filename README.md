 Projeto: Análise de Ataques Brute Force com Medusa

 Descrição do Projeto
Projeto acadêmico para estudo e análise de ataques de força bruta utilizando a ferramenta Medusa.

 Objetivos
- [ ] Compreender o funcionamento de ataques brute force
- [ ] Implementar e configurar a ferramenta Medusa
- [ ] Realizar testes controlados em ambiente seguro
- [ ] Analisar resultados e medidas de proteção
 Instalação

```bash
git clone https://github.com/seu-usuario/medusa-brute-force-project
cd medusa-brute-force-project

# Instalar dependências
pip install -r requirements.txt

# Configurar ambiente
python src/setup_environment.py
